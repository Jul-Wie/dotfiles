# Dotfiles
This repo contains my most important dotfiles, as well as instructions for myself on how to replicate my setup whenever i get a new OS.
## List of stuff

- OS: arch based
- DM: ly
- WM: awesome-git
  
- Music: spotify-tui + spotifyd
- Browser: librewolf
- Cloud sync: rclone
- Documents: libreoffice
- File manager: Thunar
- Term: Alacritty
